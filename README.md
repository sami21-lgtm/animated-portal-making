# 🌟 Premium Glassmorphism Animated Card

A high-quality, responsive glassmorphism login and content card inspired by modern UI trends. This project features dynamic background animations, a 3D tilt effect, and a secure credential check.

---

## 🚀 Features
- **Accurate Glassmorphism:** Uses `backdrop-filter` for a realistic frosted glass effect.
- **Neon Background:** Animated floating spheres with vibrant pink, purple, and cyan gradients.
- **Fully Responsive:** Optimized for both **Mobile** (touch-friendly) and **Desktop** (mouse-friendly).
- **3D Tilt Interaction:** The card reacts to your cursor or finger movement.
- **Secure Access:** Built-in JavaScript logic for name and password verification.

---

## 📂 File Structure
*   `index.html` - The structural layout.
*   `style.css` - Styles, neon colors, and background animations.
*   `script.js` - Authentication logic and 3D tilt interactions.

---

## 🛠️ Setup Instructions

1.  **Clone or Download:** Save all three files into a single folder.
2.  **Configuration:** 
    Open `script.js` and edit the following variables to set your own credentials:
    ```javascript
    const name = "Admin";
    const pass = "1234";
    ```
3.  **Run:** Open `index.html` in any modern web browser (Chrome, Safari, Edge).

---

## 📱 Mobile vs Laptop Support
- **Laptop:** The card tilts based on `mousemove` events.
- **Mobile:** The card uses `touchmove` events to ensure a smooth experience on smartphones. 
- **Auto-Scale:** The UI uses `max-width` and `viewport` meta tags to ensure it looks perfect on iPhone, Android, and 4K monitors.
